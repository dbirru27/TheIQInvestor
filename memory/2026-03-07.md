## EWROS — Exponential Weighted Relative Outperformance Score (NEW INVENTION)
- Dan's idea! Novel metric combining exponential weighting + relative daily alpha
- Formula: For each of last 63 trading days:
  - daily_alpha = stock_return - SPY_return
  - weight = e^(-0.03 × days_ago)  (λ=0.03, ~23 day half-life)
  - EWROS_raw = Σ(daily_alpha × weight)
  - Rank all 1,015 stocks → percentile 1-99
- Trend: computes EWROS at current window vs 21 days back, shows ▲/▼ direction
- Popup on click: score now vs 1mo ago, 3mo return vs SPY, win rate, avg daily alpha, best/worst day
- All stats computed during single scan (no extra API calls, ~20 seconds for 1,015 stocks)
- Replaces old IBD RS Rating throughout the site
- Wired into daily scan pipeline (scan_all.py → ewros.py → generate_watchlist.py)

## EWROS vs Rotation Analysis
- Correlation: 0.61 (related but not same)
- They measure different things:
  - EWROS = who IS outperforming (momentum)
  - Rotation = who is SETTING UP to outperform (breakout readiness)
- High EWROS + High Rotation = best entries (momentum + fresh setup)
- High EWROS + Low Rotation = already ran, extended
- Low EWROS + High Rotation = early signal, hasn't moved yet
